# FizeIO
IO操作类库。

FizeIO 是对I/O(输入输出对象)的功能性类库，实现文件夹、文件、缓冲区、流、上传等IO类管理操作功能。
FizeIO 作为基础功能类库，众多 Fize 项目依赖 FizeIO ，同时欢迎您使用 Fize。
FizeIO 有非常完善的参考文档，其代码结构也经过精雕细琢，精炼而实用。
